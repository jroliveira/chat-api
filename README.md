# chat-api

Api do widget de chat feito em nodejs com express e socket.io.

### Installing

* `git clone https://github.com/jroliveira/chat-api.git`
* `npm install`

### How to use it

* `npm start`

##### In action

Se você quer ver rodando clique [aqui](http://jroliveira.net/chat-sample/)
